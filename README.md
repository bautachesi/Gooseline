# Gooseline 🦆

Una aplicación de mensajería instantánea privada y amigable con los gansos.

## Características

- ✅ Mensajería en tiempo real con WebSocket
- ✅ Autenticación con JWT
- ✅ Cifrado de contraseñas
- ✅ Subida de avatares
- ✅ Base de datos SQLite
- ✅ Interfaz responsive

## Requisitos

- Python 3.8+
- Node.js (opcional)

## Instalación

### Backend

```bash
# Crear ambiente virtual
python -m venv venv

# Activar (Windows)
venv\Scripts\activate
# Activar (Mac/Linux)
source venv/bin/activate

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar servidor
python main.py
